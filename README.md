# chat-application

## Technologies Used

- **Backend**: 
  - Node.js
  - Express framework
  - WebSockets for real-time communication
- **Frontend**: 
  - Next.js (latest App Router)
  - Tailwind CSS for styling
- **Languages**: 
  - JavaScript
  - TypeScript

## Frontend
 - npm i
 - npm run dev

## Backend
 - npm i
 - npm run dev
